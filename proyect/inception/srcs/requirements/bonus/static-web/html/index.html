Bonus Inception

Servicio web estático como bonus del proyecto Inception (42 Madrid)

Resumen

¿Qué es esto?

Un sitio HTML/CSS servido desde un contenedor Docker con Nginx, accesible solo por proxy reverso.

Características

Lo que incluye

  • Contenedor Docker con Nginx
  • Contenido estático y de solo lectura
  • Acceso mediante proxy reverso
  • Red Docker personalizada

Tecnologías

Herramientas usadas

  • Docker & Docker Compose
  • Nginx
  • HTML5 / CSS3
  • Makefile
  • HTTP

Configuración Técnica

Contenedor

Basado en nginx:alpine, puerto 80, con volumen para archivos estáticos y configuración personalizada.

Estructura del proyecto

  • src/ – Código fuente
  • conf/ – Configuración
  • docker/ – Contenedores
  • Makefile – Automatización

Despliegue

Pasos

  1. Construir imagen Docker
  2. Configurar volúmenes
  3. Desplegar con Docker Compose
  4. Configurar proxy reverso
  5. Verificar servicio

Comandos útiles

  • make build – Construir
  • make up – Iniciar
  • make down – Detener
  • make logs – Ver logs
