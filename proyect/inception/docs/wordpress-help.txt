+-------------------------------+-----------------------------------------------------------------------------------------+
|           ACCIÓN              |                                      COMANDO                                            |
+-------------------------------+-----------------------------------------------------------------------------------------+
| Acceder al contenedor         | docker exec -it wordpress sh                                                            |
+-------------------------------+-----------------------------------------------------------------------------------------+
| Ver archivos del sitio        | ls /var/www/html                                                                        |
+-------------------------------+-----------------------------------------------------------------------------------------+
| Ver contenido wp-config       | cat /var/www/html/wp-config.php                                                         |
+-------------------------------+-----------------------------------------------------------------------------------------+
| Ver usuarios de WordPress     | wp user list --path=/var/www/html                                                       |
+-------------------------------+-----------------------------------------------------------------------------------------+
| Crear nuevo usuario           | wp user create nombre email --role=editor --user_pass=contraseña --path=/var/www/html   |
+-------------------------------+-----------------------------------------------------------------------------------------+
| Cambiar rol de usuario        | wp user update nombre --role=administrator --path=/var/www/html                         |
+-------------------------------+-----------------------------------------------------------------------------------------+
| Eliminar usuario              | wp user delete nombre --reassign=1 --path=/var/www/html                                 |
+-------------------------------+-----------------------------------------------------------------------------------------+
| Ver plugins instalados        | wp plugin list --path=/var/www/html                                                     |
+-------------------------------+-----------------------------------------------------------------------------------------+
| Activar plugin                | wp plugin activate nombre-plugin --path=/var/www/html                                   |
+-------------------------------+-----------------------------------------------------------------------------------------+
| Desactivar plugin             | wp plugin deactivate nombre-plugin --path=/var/www/html                                 |
+-------------------------------+-----------------------------------------------------------------------------------------+
| Actualizar WordPress          | wp core update --path=/var/www/html                                                     |
+-------------------------------+-----------------------------------------------------------------------------------------+
| Actualizar plugins            | wp plugin update --all --path=/var/www/html                                             |
+-------------------------------+-----------------------------------------------------------------------------------------+
| Ver versión de WordPress      | wp core version --path=/var/www/html                                                    |
+-------------------------------+-----------------------------------------------------------------------------------------+
